# 🏆 Sistema da Copa do Mundo (CRUD MVC)

## 📝 Sobre o Projeto
Este projeto foi desenvolvido como desafio prático durante o curso Técnico em Desenvolvimento de Sistemas. O objetivo foi construir um sistema completo de gerenciamento de seleções da Copa do Mundo utilizando PHP, MySQL e arquitetura MVC.

O sistema permite cadastrar, visualizar, editar e excluir seleções, seguindo boas práticas de desenvolvimento e separação de responsabilidades com o padrão MVC (Model-View-Controller).

---

## 🚀 Tecnologias Utilizadas

- PHP
- MySQL
- PDO
- HTML
- CSS
- Arquitetura MVC

---

## ⚙️ Funcionalidades

- Cadastro de seleções
- Listagem de seleções
- Edição de seleções
- Exclusão de seleções
- Sistema MVC organizado
- Proteção contra SQL Injection com PDO

---

## 📊 Funcionalidades em Desenvolvimento

- Cadastro de jogadores por seleção
- Dashboard com estatísticas
- Filtro por grupo
- Upload de bandeira das seleções

## 🛠️ Como Executar o Projeto

1. Clonar o repositório
2. Criar banco de dados MySQL
3. Importar a tabela selecoes
4. Configurar arquivo database.php
5. Rodar o servidor PHP

```bash
php -S localhost:8080
```

Abrir no navegador:

```
http://localhost:8080
```
